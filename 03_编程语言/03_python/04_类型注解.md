# 类型注解

目录：

- [函数类型注解](#函数类型注解)

## 基本介绍

- **介绍**：类型注解是 `Python` 中的一种语法特性，用于明确标识变量、函数参数和返回值的数据类型，从而使代码更清晰、更安全、更易维护
- **案例**：
```python
a: int = 695  
score: float = 98.5  
hobby: str = "Python"  
flag: bool = True  
pic: None = None  
  
names: list[str] = ["A","B","C","D"]  
phones: set[str] = {"159236456","026674555"}  
options: dict[str,int] = {"count":0,"total":0}  
goods: tuple[str,int,int] = ("手机",5999,1)
```
**注意**：类型注解只是起到**语法提示**的作用，并不会影响程序运行的结果，非**强制约束**（Python 为动态类型语言）

## 函数类型注解

- **介绍**：主要为函数的参数和返回值添加类型注解
- **案例**：
```python
def calc(scores: list[int]) -> float
	return sum(scores) / len(scores)
	
# 上面的 list[int] 是参数的类型，-> float 是返回值的类型
```

**注意**：对于团队协作开发和长期维护的项目，推荐**使用类型注解**
