# DFS

## 什么是 DFS

我们不妨置身于一个迷宫之中，假设我们想要找到从入口到出口的路径。那么，那么 DFS 就是一个人沿着一条路径一直走到底，如果走不通了就回到上一个路口，换一条路继续走，直到找到出口为止。

## DFS 的基本框架：树

DFS 的基本框架可以用树来表示。树的每一个节点代表一个状态，每一条边代表从一个状态到另一个状态的转换。DFS 的过程就是从根节点开始，沿着树的边一直向下走，直到到达叶子节点或者找到目标状态为止。如果走到叶子节点还没有找到目标状态，就回到上一个节点，换一条边继续走。

### 树的 DFS 遍历

下面是一个用递归实现的树的 DFS 遍历的代码示例：

```cpp
void dfs(TreeNode* node) {
    if (node == nullptr) {
        return;
    }
    // 访问当前节点
    cout << node->val << " ";
    // 递归遍历左子树
    dfs(node->left);
    // 递归遍历右子树
    dfs(node->right);
}
```
