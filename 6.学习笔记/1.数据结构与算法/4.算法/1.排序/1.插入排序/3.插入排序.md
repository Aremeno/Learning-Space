# 插入排序

插入排序（Insertion Sort）是一种简单的排序算法，它的工作原理类似于我们整理扑克牌手牌的方式。在未排序部分中找一个基准元素，对已排序部分从后向前扫描，找到合适的位置插入该基准元素。重复此过程，直到所有元素均排序完毕。

## 算法步骤

1. 从数组的第二个元素开始，假设第一个元素已经排序。
2. 取出当前元素（称为“基准”）。
3. 将基准与已排序部分的元素进行比较，从后向前扫描已排序部分。
4. 如果已排序部分的元素大于基准，则将该元素向后移动一位。
5. 重复步骤4，直到找到合适的位置插入基准。
6. 将基准插入到找到的位置。
7. 重复步骤2-6，直到整个数组排序完成。

## 代码实现

```c
#include <stdio.h>

void insert_sort(int arr[],int size){
    for(int i=1;i<size;i++){
        int base = arr[i],j = i-1;
        while(j >=0 && arr[j] > base){
            arr[j+1] = arr[j];
            j--;
        }
        // j就是前一个的下标，不满足前面大的条件了，直接把它插入到该数前面就行了
        arr[j+1] = base;
    }
}
```

## 复杂度分析

- 时间复杂度：最坏情况下为O(n^2)，平均情况下为O(n^2)，最好情况下为O(n)（当数组已经有序时）。
- 空间复杂度：O(1)，因为插入排序是原地排序算法，不需要额外的存储空间。
- 稳定性：插入排序是稳定的排序算法，因为相等的元素不会改变它们的相对顺序。
