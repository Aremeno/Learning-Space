# 二叉树

二叉树是一种非线性数据结构，二叉树的基本单元是节点，每个节点包含值、左子节点引用和右子节点引用。

```cpp
class TreeNode {
    public:
    int val; // 节点值
    TreeNode* left; // 左子节点
    TreeNode* right; // 右子节点
    TreeNode(int x) : val(x), left(NULL), right(NULL) {}
};
```

每个节点（父节点）都有两个引用，分别指向它的左子节点和右子节点。将该节点的左子节点及其所有子节点组成的结构称为左子树，右子节点及其所有子节点组成的结构称为右子树。

## 常见术语

- 根节点（Root Node）：树的顶层节点，无父节点。
- 叶节点（Leaf Node）：最后的节点，无子节点。
- 边（Edge）：节点引用（指针），即父节点与子节点之间的连接。
- 层级（Level）：节点所在的层次。
- 度（Degree）：节点的子节点数量。
- 高度（Height）：从节点到叶节点的最长路径上的边数。
- 深度（Depth）：从根节点到该节点的路径上的边数。
- 子树（Subtree）：节点及其所有后代节点组成的树。
- 森林（Forest）：多个不相连的树的集合。

## 二叉树的基本操作

### 初始化二叉树

```cpp
TreeNode* n1 = new TreeNode(1);
TreeNode* n2 = new TreeNode(2);
TreeNode* n3 = new TreeNode(3);
TreeNode* n4 = new TreeNode(4);
TreeNode* n5 = new TreeNode(5);

n1->left = n2; // 1的左子节点是2
n1->right = n3; // 1的右子节点是3
n2->left = n4; // 2的左子节点是4
n2->right = n5; // 2的右子节点是5
```

### 插入与删除节点

在二叉树中插入与删除节点可以通过修改指针来实现。

**插入节点：**

```cpp
void insertnode(TreeNodde *parent , int value) {
    TreeNode *p = new TreeNode(value);
    parent->left = p;
    p->left = n2;
}
```

**删除节点：**

```cpp
// 删除上述插入的p节点
parent->left = n2;
delete p;
```

**注意：**一般来说，二叉树的插入与删除操作通常是由一套操作配合来完成的，如二叉搜索树的插入与删除需要保持树的有序性。

## 常见二叉树的类型

1. **完美二叉树（ Perfect Binary Tree）**：说实话其实就是填满了的二叉树。
2. **完全二叉树（ Complete Binary Tree）**：除了最后一层节点外，其他层的节点都被填满，并且最后一层的节点从左到右依次排列。
3. **满二叉树（ Full Binary Tree）**：每个节点要么没有子节点，要么有两个子节点。
4. **平衡二叉树（ Balanced Binary Tree）**：任意节点的左子树和右子树的高度差不超过1。
